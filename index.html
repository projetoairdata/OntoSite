O projeto Artificial Intelligence for Aviation Data Analysis (AirData) envolve o desenvolvimento de um sistema de armazenamento e análise de dados para o setor de aviação brasileiro, focado na aviação civil, fundamentado em técnicas de Inteligência Artificial (IA).

Uma das partes integrantes do projeto é o desenvolvimento de uma ontologia formal para o setor. A ontologia funciona como uma base de conhecimento estruturada que permite aos modelos de linguagem (LLMs) compreender com precisão os conceitos, relações e regras do domínio aeronáutico. Ao integrar essa estrutura semântica, os modelos podem fornecer respostas mais precisas, contextualizadas e alinhadas com a terminologia técnica da aviação civil brasileira.

Este portal apresenta o status atual de desenvolvimento da ontologia AirData.
